## Install

`npm install --save articleforge`


## Scripts

 - **npm run build** : `rimraf dist && tsc`
 - **npm run lint** : `gts lint`
 - **npm run clean** : `gts clean`
 - **npm run compile** : `tsc`
 - **npm run fix** : `gts fix`
 - **npm run readme** : `node ./node_modules/.bin/node-readme`
